制作方法（方便喜欢自定义或自己更新版本的人使用）：

制作环境：Windows

使用工具

  • sphinx-doc 这个需要先安装 python
  • Html Help Workshop
  • Phalcon 文档项目副本

制作步骤

  1. 安装 python，安装完的时候勾选添加到 环境变量，或者手动添加
  2. 进入CMD，执行：easy_install sphinx 来安装 sphinx-doc
  3. 进入 Phalcon 文档项目副本，进入到对应需要编译的语言，我这里是中文 zh
  4. 不选定文件点击鼠标右键，选定在“在此处打开命令窗口”
